Case Studies

Rise Broadband Call Center Dashboards

Loveland, Colorado

Weekly Users: ~175 Employees

Rise Broadband's busy call center used the Five9 IVR system to handle calls. Unfortunately, the data visualization and queue monitoring capabilities of Five9 were highly limited.

We built a custom web application to consolidate the Five9 data and bring real-time statistics to all contact center employees, with features including:

  • - Views for representatives, supervisors and managers
  • - Real-time sales tracking
  • - Live maps showing number of callers by area code
  • - Integration with Rise's existing Business Intelligence systems

Women2Women Application Manager

Greeley, Weld County, Colorado

Weekly Users: ~25

Women2Women had an existing web application used by Weld County agencies to help women apply for financial assistance. We were brought in to add new features and fix some long-standing bugs.

  • - Helpful messages when application forms are filled in incorrectly, reducing troubleshooting calls to the Women2Women team
  • - A Comment System on applications to streamline communications while voting
  • - Security improvements resulting in an "A" SSL rating from SSL Labs and better protection of sensitive data
  • - CSV reporting to allow application data to be exported and manipulated in Excel
